# Task 1 - HTML Structure and Basic Server Interaction

## Objective

This project demonstrates a simple Node.js and Express.js application using EJS for server-side rendering. It allows users to submit a registration form and displays the submitted information on another page.

---

## Technologies Used

- Node.js
- Express.js
- EJS
- HTML5

---

## Features

- User Registration Form
- Server-side Rendering using EJS
- Handles Form Submission
- Displays Submitted Data
- Simple and Easy-to-Understand Project

---

## Project Structure

Task1/
│── server.js
│── README.md
└── views/
    ├── index.ejs
    └── result.ejs

---

## How to Run the Project

1. Install Node.js.
2. Open the project folder in VS Code.
3. Open the terminal.
4. Run:

```bash
node server.js
```

5. Open your browser and visit:

```
http://localhost:3000
```

---

## Output

- User Registration Form
- Displays submitted Name, Email, Phone Number, and Age on the result page.
